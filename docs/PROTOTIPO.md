Fluxo

Início do protótipo

Tela inicial (Login / Bem-vinda)

Primeiro ponto de partida do fluxo.
A tela inicial obriga o usuário a logar ou criar conta.

Caminhos possíveis
Ação do usuário	        Fluxo resultante / tela de destino
Clicar em “Criar conta”	Vai para a Tela de Cadastro (registro)
Inserir dados e clicar “Entrar”	Vai para o Catálogo de Arranjos
Após cadastro e confirmação	Também vai para o Catálogo de Arranjos
Catálogo de Arranjos (Tela principal pós-login)

Exibe lista ou grid de arranjos/flores com imagem, nome e preço.

Permite buscar e/ou filtrar (por categoria, promoção etc).

Cada cartão de produto é clicável.

Ações possíveis no catálogo

Clicar em um produto → leva à Tela de Detalhes do Produto

Clicar ícone de carrinho → leva ao carrinho 

Detalhes do Produto

Mostra imagem, nome, descrição, preço, opções (tamanho, personalização).

Botões disponíveis: “Adicionar ao Carrinho” e “Comprar Agora”.

Ações possíveis

Adicionar ao Carrinho → item adicionado (badge/indicador do carrinho atualizado)

Comprar Agora → leva direto ao Checkout/Pedido (Carrinho + agendamento)

Carrinho / Checkout / Finalização de Pedido

Usuário vê itens adicionados, pode revisar quantidade/subtotal.

Preenche dados de entrega / escolhe data e período (manhã, tarde, noite).

Clica em Confirmar Pedido → surge uma tela de Confirmação (overlay ou nova tela) com mensagem de sucesso e número de pedido/recibo.

Depois da confirmação

Opção de voltar ao catálogo ou sair / encerrar protótipo

Fluxo completo de compra finalizado
